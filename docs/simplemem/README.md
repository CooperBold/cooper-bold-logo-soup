# SimpleMem local store

Committed memory file: **`memories.json`** in this directory.

- Configure via `.env`: `SIMPLEMEM_LOCAL_DIR=docs/simplemem`, `SIMPLEMEM_NAMESPACE=logo-soup-wp-plugin`
- Import session log: `python3 simplemem_cli.py import-ai-session --path AI_SESSION_MEMORY.md`
- Do not store secrets or PII.
